Workshop Resources

1. Installation Guide

Prior to the workshop, please take a few moments to prepare your computer by installing the necessary software dependancies. We will be using a tool that requires both Node version 4 or higher and Gulp 4 (currently in Alpha).

The easiest way to install Node on macOS is via Homebrew. At the terminal:

brew install node

To install Gulp 4, you’ll first want to remove any older versions of Gulp:

npm uninstall -g gulp

You can then install Gulp 4 with:

npm install gulpjs/gulp-cli -g

2. Pattern Lab Project

We’ll be using Pattern Lab to build out our modular components and create a design system for our sample project. The following ZIP file contains both the starting and completed version of our project:

{% include svg/file.svg %} workshop-patternlab.zip

Inside the PatternLab-00 directory, first install NPM dependencies:

npm install

You can then start the project using the following command:

gulp patternlab:serve
